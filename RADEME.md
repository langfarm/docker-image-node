## Node 环境

* 基本 alinux3（Alibaba Cloud Linux 3） 版本 3.9
* 包括 fnm
* fnm 安装 node

## 特性：
- 默认 node 源： 华中科学大学 node 源 https://mirrors.ustc.edu.cn/node/
- 用 fnm 来安装 node
- npm 源：设置淘宝源 https://registry.npmmirror.com
